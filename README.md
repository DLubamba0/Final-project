# Basketball Game & Tournament Management REST API

This project is a backend REST API for managing basketball teams, players, and games. It includes CRUD operations, JWT authentication, role-based authorization, Sequelize ORM, SQLite database storage, and basic testing.

## Features

- Node.js and Express REST API
- Sequelize ORM with SQLite database
- Teams, Players, and Games resources
- Full CRUD operations
- JWT authentication
- Role-based authorization
- Admin and Player user roles
- Basic error handling
- Unit testing with Jest and Supertest
